# 인성의 웹페이지 📚

교육용 웹 애플리케이션 모음 사이트입니다. 딥러닝 학습 게임과 학급 자리 배치 도구를 제공합니다.

## 🎯 프로젝트 소개

학생들이 인공지능과 교실 관리를 재미있게 배울 수 있도록 설계된 인터랙티브 웹 애플리케이션입니다.

### 주요 기능

1. **🤖 AI 트레이너 게임** (`deeplearning.html`)
   - 딥러닝의 기초 개념을 게임으로 학습
   - 지도 학습(Supervised Learning) 체험
   - 도형 인식 AI 훈련 시뮬레이션
   - 4단계 레벨 시스템과 점수/별 보상

2. **🪑 학급 자리 배치 도구** (`seatingarrangement.html`)
   - 학생 명단 관리 (자동 이름 생성 지원)
   - 드래그 앤 드롭으로 책상 배치
   - 랜덤 자리 배정 기능
   - 반응형 디자인 (데스크탑/모바일)

## 🎮 사용 방법

### AI 트레이너 게임

1. 화면에 표시되는 도형들 중 목표 도형 선택
2. "AI 학습시키기" 버튼 클릭
3. 정답을 맞추면 AI가 학습하고 점수 획득
4. 각 레벨의 목표 정답 수를 달성하면 다음 레벨로 진행
5. 4개 레벨을 모두 완료하면 최종 점수 확인

### 학급 자리 배치 도구

1. **학생 수 입력**: 학급의 학생 수를 입력
2. **명단 확인**: 자동 생성된 이름 확인 및 수정
3. **자리 배치**: 
   - 책상을 드래그하여 원하는 위치로 이동
   - "책상 추가" 버튼으로 추가 책상 배치
   - "배치 초기화"로 기본 6x4 레이아웃 복원
4. **자리 배정**: "자리 배정하기" 버튼으로 랜덤 배정
