# Inseong Jeon's Portfolio Website

This repository hosts the personal website and portfolio of Inseong Jeon.

## File Structure

- **index.html**: Main landing page leading to portfolio and CV.
- **pages/**: Contains sub-pages.
  - **portfolio.html**: The detailed portfolio page (previously the main page).
  - **cv.html**: Curriculum Vitae.
  - **dqn-demo.html**: Deep Q-Network visualization.
  - **microrobot.html**: Micro Robot game.
  - **stackburger.html**: StackBurger project.
  - **attendance.html**: Attendance tracking.
- **css/**: Global styles.
  - **style.css**: Main stylesheet with dark mode support.
- **js/**: Global scripts.
  - **main.js**: Handles dark mode, language toggle, and mobile menu.
- **AlligatorChess/**: Game directory (root level).
- **astar-pathfinding/**: Algorithm demo directory (root level).
- **simplequiz/**: Quiz application directory (root level).

## Features

- **Responsive Design**: Works on mobile and desktop.
- **Dark Mode**: User preference saved in local storage.
- **Bilingual Support**: Toggle between English and Korean.
- **Interactive Demos**: Links to various JavaScript projects.

## Tech Stack

- HTML5, CSS3, JavaScript (Vanilla).
- No external frameworks (except Google Fonts).
